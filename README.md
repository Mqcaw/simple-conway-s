# simple-conway
Conway's game of life, implemented in c using SDL libraries for display.\

## Keybinds
`w/a/s/d` - move cursor\
`r` - random distribution \
`c` - clear grid\
`t` - load sierpinskies triangle demo\
`space` - pause/play\
`enter` - toggle current cell state\
`right_arrow` - next frame\
